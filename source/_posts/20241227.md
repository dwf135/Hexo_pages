2024.10.11 16:07 修复了重启无法自动启动的问题（多生成一行[Network],启动不了删除一个Network）

```
wget -N https://raw.githubusercontent.com/qqrrooty/EZrealm/main/realm.sh && chmod +x realm.sh && ./realm.sh
```
再次启动脚本


```
./realm.sh
```

如若更新脚本，请先删除脚本(不需要卸载)，再运行安装脚本
```
rm realm.sh
```
